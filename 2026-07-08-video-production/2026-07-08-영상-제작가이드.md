# Codex Subagent 구현 데모 MP4 제작가이드

## 최종 산출물

- `2026-07-08-codex-subagent-구현데모.mp4`: 최종 MP4 영상
- `2026-07-08-codex-subagent-구현데모-내레이션.mp3`: 한국어 남자 AI 음성 내레이션
- `2026-07-08-codex-subagent-구현데모-자막.srt`: 영상 자막 파일
- `2026-07-08-codex-subagent-구현데모-제작정보.json`: 장면, 음성, 길이, 파일 크기 정보

## 제작 방식

1. 장면별 설명 화면을 이미지로 렌더링합니다.
2. `ko-KR-HyunsuMultilingualNeural` 남성 음성으로 한국어 내레이션을 생성합니다.
3. 장면 이미지와 내레이션을 합쳐 MP4로 렌더링합니다.
4. 핵심 자막은 영상 하단에 직접 넣고, 별도 SRT 파일도 함께 저장합니다.

## 다시 렌더링하는 방법

작업 폴더 루트에서 아래 명령을 실행합니다.

```powershell
py -3 2026-07-08-video-production\2026-07-08-render-video.py
```

## 영상에서 보여주는 핵심 메시지

- 서브에이전트는 역할별로 작업을 나누는 방식입니다.
- 기획, 평가, 개발, 위험 점검, 디자인을 분리하면 결과 비교가 쉬워집니다.
- 메인 스레드는 결과를 모으고 최종 판단을 내립니다.
- 최종 산출물은 HTML 화면 하나가 아니라, 프롬프트, 가이드, 테스트, Git 흐름까지 포함한 재사용 템플릿입니다.

## 공개 저장소 주의

영상에는 실제 고객명, 계정 토큰, API 키, 개인 이메일을 넣지 않습니다. 데모 데이터는 샘플 문구로만 구성합니다.
