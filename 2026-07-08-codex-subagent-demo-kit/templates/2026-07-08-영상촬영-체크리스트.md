# 영상 촬영 체크리스트

## 촬영 전

- [ ] 프로젝트 폴더가 준비되어 있는지 확인합니다.
- [ ] 원격 저장소 주소를 확인합니다.
- [ ] 공개되면 안 되는 파일이나 비밀값이 없는지 확인합니다.
- [ ] 사용할 subagent 역할을 미리 정리합니다.
- [ ] 데모에서 보여줄 최종 산출물 형식을 정합니다.

## 촬영 중

- [ ] 처음에 목표를 한 문장으로 말합니다.
- [ ] subagent를 왜 쓰는지 설명합니다.
- [ ] 각 subagent에게 맡긴 일이 무엇인지 보여줍니다.
- [ ] 기획자와 평가자의 결과 차이를 보여줍니다.
- [ ] 개발자, 보조 개발자, 디자이너가 병렬로 일하는 장면을 보여줍니다.
- [ ] 최종 결과를 메인 Codex가 통합하는 장면을 보여줍니다.

## 촬영 후 확인

- [ ] 생성된 파일 목록을 보여줍니다.
- [ ] 백업 파일 위치를 보여줍니다.
- [ ] Git 커밋 결과를 보여줍니다.
- [ ] 푸시 성공 또는 실패 결과를 설명합니다.
- [ ] 다른 프로젝트에 재사용하는 방법을 말합니다.

## 자막으로 쓰기 좋은 문장

```text
Codex subagent는 AI에게 일을 더 많이 시키는 기능이 아니라,
일을 더 잘 나눠 맡기는 방식입니다.
```

```text
메인 대화는 판단과 통합에 집중하고,
subagent는 탐색, 평가, 구현, 점검 같은 역할을 맡습니다.
```

```text
좋은 데모는 설명이 긴 데모가 아니라,
화면에서 차이가 바로 보이는 데모입니다.
```
