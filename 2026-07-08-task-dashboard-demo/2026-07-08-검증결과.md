# 작업 관리 대시보드 데모 검증결과

## 검증 항목

- 상태 계산 테스트
- 브라우저 첫 화면 렌더링
- 상태 변경 후 KPI 갱신
- 데스크톱/모바일 화면 폭 확인
- 공개 저장소에 민감정보 포함 여부 확인

## 검증 기록

### 1. 상태 계산 테스트

명령:

```powershell
node 2026-07-08-task-dashboard-demo\tests\2026-07-08-dashboard-state.test.mjs
```

결과:

```text
dashboard-state tests passed
```

### 2. 브라우저 렌더링과 상호작용

확인 URL:

```text
http://127.0.0.1:8787/2026-07-08-task-dashboard-demo/2026-07-08-task-dashboard.html
```

확인 결과:

- 첫 화면 제목 렌더링 정상
- 전체 작업 수 6개 표시 정상
- `공개 저장소 보안 점검` 작업을 `위험`에서 `완료`로 변경 시 완료율 `17% -> 33%`로 갱신
- 같은 변경 후 활성 위험 수 `1 -> 0`으로 갱신
- 활동 로그에 상태 변경 내용 반영
- 모바일 폭 390px에서 가로 넘침 없음
- 브라우저 콘솔 오류 없음

### 3. 공개 저장소 점검

- 샘플 데이터만 사용
- 실명, 이메일, API 키, 토큰 없음
- 실제 업무 데이터 없음
