# 작업 관리 대시보드 데모 실행가이드

## 목적

이 데모는 Codex 서브에이전트가 역할별로 만든 산출물을 하나의 화면으로 통합해 보여주는 촬영용 대시보드입니다.

핵심 장면은 작업 카드의 상태를 바꿀 때 다음 요소가 동시에 갱신되는 모습입니다.

- 상단 KPI
- 완료율 진행률 바
- 위험 판단 목록
- 선택 작업 상세
- 활동 로그

## 실행 방법

이 폴더에서 간단한 로컬 서버를 실행한 뒤 HTML을 엽니다.

```powershell
py -3 -m http.server 8787
```

브라우저에서 아래 주소를 엽니다.

```text
http://127.0.0.1:8787/2026-07-08-task-dashboard-demo/2026-07-08-task-dashboard.html
```

## 촬영 순서

1. 사이드바에서 역할별 서브에이전트 스레드가 나뉜 모습을 보여줍니다.
2. 메인 스레드에서 역할별 결과를 통합한다고 설명합니다.
3. 대시보드 첫 화면을 엽니다.
4. `공개 저장소 보안 점검` 작업을 `위험`에서 `완료`로 바꿉니다.
5. 완료율이 올라가고 활성 위험 수가 줄어드는 장면을 보여줍니다.
6. “메인 Codex는 통합 판단을 하고, 서브에이전트는 역할별 산출물을 만든다”고 정리합니다.
